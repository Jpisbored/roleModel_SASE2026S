# How to Run Dev and Test Environment

## Dependencies
You must have either
- A computer with flutter set up on it and an android emulator
- A computer with flutter and an andoid phone with development mode turned on
- The APK pulled from this project and thrown in either an emulator or an android phone with the above requirements

Once the app has been placed on an emulator or functioning android, everything else works as intended through the usual app interaction 
**Server is only active during peak hours so the "New Session" feature is unavailable while work is not in progress at this time
